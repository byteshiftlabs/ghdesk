# Roadmap

This document outlines the planned features and improvements for ghdesk.

## Current Version: v0.1.0 ✅

- ✅ Repository management (clone, browse, sync)
- ✅ Pull request viewing and management
- ✅ Issue browsing and filtering
- ✅ Git operations (commit, push, branch management)
- ✅ Multiple theme support (Light, Dark, Dracula, Monokai, Nord)
- ✅ Collapsible sidebar with expandable directory tree
- ✅ Diff viewer with syntax highlighting

## v0.2.0 - Release & Tag Management (Planned)

Focus: Enhanced version control and release workflows

- [ ] **Release Management**
  - [ ] View existing releases for repositories
  - [ ] Create new releases with title and notes
  - [ ] Edit release descriptions
  - [ ] Delete releases
  - [ ] Upload/download release assets
  - [ ] Mark releases as draft or prerelease
  - [ ] Auto-generate release notes from commits

- [ ] **Tag Management**
  - [ ] List all tags in repository
  - [ ] Create annotated and lightweight tags
  - [ ] Delete local and remote tags
  - [ ] View tag details and associated commits
  - [ ] Push/pull tags to/from remote
  - [ ] Tag comparison and diff views

- [ ] **Version Tracking**
  - [ ] Semantic version parsing and validation
  - [ ] Version timeline visualization
  - [ ] Compare versions side-by-side

## v0.3.0 - Enhanced Collaboration (Planned)

Focus: Improved team collaboration features

- [ ] **Pull Request Enhancements**
  - [ ] Inline code review comments
  - [ ] Review approval/request changes workflow
  - [ ] Draft pull request support
  - [ ] PR templates

- [ ] **Issue Management**
  - [ ] Create and edit issues
  - [ ] Issue templates
  - [ ] Milestone management
  - [ ] Label management
  - [ ] Issue assignment


## Contributing

We welcome contributions! If you'd like to work on any of these features:

1. Check the [Issues](https://github.com/byteshiftlabs/ghdesk/issues) page for related discussions
2. Comment on an issue or create a new one to discuss your approach
3. Fork the repository and create a feature branch
4. Submit a pull request following our [Contributing Guidelines](CONTRIBUTING.md)

## Feedback

Have ideas for features not listed here? Please:
- Open an issue with the `enhancement` label
- Start a discussion in the [Discussions](https://github.com/byteshiftlabs/ghdesk/discussions) tab
- Reach out to the maintainers
