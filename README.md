# ghdesk

A desktop GUI for managing GitHub repositories and pull requests.

## Quick Start

```bash
git clone https://github.com/byteshiftlabs/ghdesk.git
cd ghdesk
pip install -r requirements.txt
python main.py
```

## Requirements

- Python 3.8+
- GitHub CLI (`gh`) installed and authenticated
- See [requirements.txt](requirements.txt) for dependencies

## What It Does

Provides a graphical interface for:
- Browsing local repositories with expandable file tree
- Viewing and managing pull requests
- Creating commits and pushing changes
- Repository operations (clone, sync, settings)
